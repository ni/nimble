CSS Code Examples

Custom Sizing:

.size-demo::part(control) {
    max-width: 200px;
    max-height: 100px;
    overflow: auto;
}

Custom Appearance:

.appearance-demo::part(control) {
    border-radius: 12px;
    border: 2px solid #007acc;
    background-color: #f0f8ff;
    box-shadow: 0 4px 12px rgba(0, 122, 204, 0.3);
}

Error Styling:

.error-demo::part(control) {
    border: 2px solid #d32f2f;
    background-color: #ffebee;
    border-radius: 6px;
    max-width: 300px;
}
