## Styling

The `nimble-table` should be sized explicitly or sized to fill the space of a parent container. The `nimble-table` does not currently support being styled with `height: auto`. The ability to auto size the table is tracked with [issue 1624](https://github.com/ni/nimble/issues/1624).
