## Usage

### React

When assigning a ref created with `useRef` to a `NimbleDrawer`, make sure to use the `fromDrawerRef` helper to avoid type issues due to the generic component reference. For example:

```tsx
<NimbleDrawer ref={fromDrawerRef(drawerRef)}></NimbleDrawer>
```

For more information see [ni/nimble#2784](https://github.com/ni/nimble/issues/2784).
