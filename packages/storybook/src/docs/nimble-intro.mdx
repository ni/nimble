# Nimble Design System

<img src={nimbleLogo} width="100px" />

## Overview

The [Nimble Design System](https://nimble.ni.dev/) is an easy to use collection
of well-designed, styled components that can be used in any NI web application.

Outside the core design system, the
[Spright component collection](https://github.com/ni/nimble/tree/main/packages/spright-components/README.md)
is home to more complex, experimental, or application-specific components.

Refer to the [Nimble GitHub repository](https://github.com/ni/nimble) for
documentation on using or contributing to the component libraries. To add or
update component design documentation, refer to the
[documentation guide](https://github.com/ni/nimble/tree/main/packages/storybook/CONTRIBUTING.md).

See the <a href="./example-client-app" target="_blank">example Angular app</a>
or <a href="./blazor-client-app/wwwroot" target="_blank">example Blazor app</a>
using the components!

See the <a href="./performance" target="_blank">component performance tests</a>.
