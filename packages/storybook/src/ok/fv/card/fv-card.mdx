The Fv card is an opinionated card layout for concise title, subtitle, description,
metadata, badge, icon, and action combinations.

Use the optional `icon`, `badges`, `footer-start`, `footer-end`, and `actions`
slots to add the supported layout regions.

`outline` is the default appearance. `block` uses a filled surface. `interaction-mode`
switches between the static card shell and the interactive card-button shell.
